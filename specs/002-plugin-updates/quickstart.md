# Quickstart

This feature introduces two new commands to the Obsidian command palette.

## Check for Plugin Updates

1. Open the command palette (Cmd/Ctrl + P).
2. Type "Check for plugin updates".
3. Run the command.
4. A notification will appear with a list of plugins that have available updates.

## Update All Plugins

1. Open the command palette (Cmd/Ctrl + P).
2. Type "Update all plugins".
3. Run the command.
4. All plugins with available updates will be updated. You will receive notifications from Obsidian and BRAT as the updates complete.
